Inicio rápido de Java



En Java, cada aplicación comienza con un nombre de clase y esa clase debe coincidir con el nombre del archivo.

Vamos a crear nuestro primer archivo Java, llamado Main.java, que se puede hacer en cualquier editor de texto (como el Bloc de notas).

El archivo debe contener un mensaje "Hello World", que está escrito con el siguiente código:.



Main.java

public class Main {
public static void main(String[] args) {
System.out.println("Hello World");
}
}



No se preocupe si no entiende el código anterior; lo discutiremos en detalle en capítulos posteriores. Por ahora, concéntrese en cómo ejecutar el código anterior.

Guarde el código en el Bloc de notas como "Main.java". Abra el símbolo del sistema (cmd.exe), navegue hasta el directorio donde guardó su archivo y escriba "javac Main.java":



C:\Usuarios\Su nombre>javac Main.java



Esto compilará su código. Si no hay errores en el código, el símbolo del sistema lo llevará a la siguiente línea. Ahora, escribe "Java Main" para ejecutar el archivo:



C:\Usuarios\Tu nombre>java Principal



La salida debe leer:



Hello World
